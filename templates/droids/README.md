# Oh My Droid Agents

This directory contains droid agent definitions for the oh-my-droid plugin.

Agents will be automatically loaded by Factory AI Droid from this directory.

## Agent Tiers

Agents are organized into three tiers based on model capability:

- **LOW (Haiku)**: Fast, cost-effective for simple tasks
- **MEDIUM (Sonnet)**: Balanced performance for standard work
- **HIGH (Opus)**: Maximum capability for complex reasoning
